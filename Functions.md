# Functions

Functions are reusable blocks of code that perform a specific task. They allow you to break down your code into smaller, more manageable pieces, making it easier to read and maintain. In Python, you can define a function using the `def` keyword, followed by the function name and parentheses.

Here is an example of a simple function that takes two numbers as input and returns their sum:

```python
def add_numbers(a, b):
    return a + b

# Example usage
result = add_numbers(5, 3)
print(result)  # Output: 8
```
