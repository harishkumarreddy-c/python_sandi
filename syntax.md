# 🧮 Python Syntax :

1. **Indentation**: Python uses indentation to define blocks of code. Consistent indentation is crucial for the correct execution of the program.

```pythonif True:
    print("This is indented")
```

2. **Comments**: Comments are used to explain code and are ignored by the interpreter. Single-line comments start with `#`, while multi-line comments can be enclosed in triple quotes (`'''` or `"""`).

```python # This is a single-line comment

'''
This is a multi-line comment
'''
```
