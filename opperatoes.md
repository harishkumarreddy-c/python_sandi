# Operators:

Operators are symbols that perform operations on variables and values. In Python, there are several types of operators, including:

1. Assignment Operators: Used to assign values to variables.
   - `=`: Assigns a value to a variable (e.g., `x = 5`).
   - `+=`: Adds and assigns (e.g., `x += 3` is equivalent to `x = x + 3`).
   - `-=`: Subtracts and assigns (e.g., `x -= 2` is equivalent to `x = x - 2`).
   - `*=`: Multiplies and assigns (e.g., `x *= 4` is equivalent to `x = x * 4`).
   - `/=`: Divides and assigns (e.g., `x /= 2` is equivalent to `x = x / 2`).

2. Arithmetic Operators: Used to perform mathematical operations.
   - `+`: Addition (e.g., `5 + 3` results in `8`).
   - `-`: Subtraction (e.g., `5 - 3` results in `2`).
   - `*`: Multiplication (e.g., `5 * 3` results in `15`).
   - `/`: Division (e.g., `5 / 3` results in `1.666...`).
   - `//`: Floor division (e.g., `5 // 3` results in `1`).
   - `%`: Modulus (e.g., `5 % 3` results in `2`).
   - `**`: Exponentiation (e.g., `5 ** 3` results in `125`).

3. Comparison Operators: Used to compare values.
   - `==`: Equal to (e.g., `5 == 5` results in `True`).
   - `!=`: Not equal to (e.g., `5 != 3` results in `True`).
   - `>`: Greater than (e.g., `5 > 3` results in `True`).
   - `<`: Less than (e.g., `5 < 3` results in `False`).
   - `>=`: Greater than or equal to (e.g., `5 >= 5` results in `True`).
   - `<=`: Less than or equal to (e.g., `5 <= 3` results in `False`).

4. Logical Operators: Used to combine conditional statements.
   - `and`: Returns `True` if both statements are true (e.g., `True and False` results in `False`).
   - `or`: Returns `True` if at least one statement is true (e.g., `True or False` results in `True`).
   - `not`: Reverses the result (e.g., `not True` results in `False`).

5. Bitwise Operators: Used to perform bit-level operations.
   - `&`: Bitwise AND (e.g., `5 & 3` results in `1`).
   - `|`: Bitwise OR (e.g., `5 | 3` results in `7`).
   - `^`: Bitwise XOR (e.g., `5 ^ 3` results in `6`).
   - `~`: Bitwise NOT (e.g., `~5` results in `-6`).
   - `<<`: Left shift (e.g., `5 << 1` results in `10`).
   - `>>`: Right shift (e.g., `5 >> 1` results in `2`).
